# Sample Presentation

## Introduction

- **Purpose** — Demonstrate the markdown-to-presentation generator
- **Features** — Progress tracking, collapsible cards, persistent state

## Getting Started

- **Step 1** — Install Python 3.7 or higher
- **Step 2** — Run the generator with your markdown file
- **Step 3** — Open the generated HTML in your browser

## Live Demo

https://example.com/demo

## Advanced Features

- **Theme Customization** — Edit CSS variables to change colors
- **Responsive Design** — Works on desktop and mobile devices
- **Offline Support** — No external dependencies, works without internet

More advanced features are coming soon.
